# Assets

**Background pattern:** The signup right panel uses `group-1.png` as a tiled background with a blue tint. Add `group-1.png` (and optionally `group-2.png`, `group-3.png`, `group-4.png`) here.

**Continue with Email button:** The mail icon is loaded from `mail.svg`. Add your mail icon here with that name (or we can update the code to match your filename).

**Check:** `http://localhost:3000/assets/group-1.png` and `http://localhost:3000/assets/mail.svg` should return the images when the files exist here.
